Privacy Policy

Last Updated: November 2024

Introduction

This Privacy Policy describes how Voice Chat Assistant for ChatGPT ("we," "our," or "the extension") collects, uses, and handles your information when you use our Chrome extension.

Information Collection and Use

API Keys

We store your ElevenLabs API key locally in your browser using Chrome's storage API. This key is never transmitted to our servers and is only used to authenticate requests directly with ElevenLabs' services.

Voice Input

  • The extension uses your device's microphone only when explicitly activated by you
  • Voice input is processed locally using the Web Speech API
  • Transcribed text is sent directly to ChatGPT through their existing interface
  • No voice recordings are stored or transmitted elsewhere

Text Processing

  • ChatGPT responses are processed through ElevenLabs' text-to-speech API
  • Text content is transmitted directly from your browser to ElevenLabs using your provided API key
  • No text content is stored on our servers

Data Storage

The extension stores only:

  • Your ElevenLabs API key (stored locally in Chrome storage)
  • Temporary audio data for playback (cleared when audio completes or extension is disabled)

Third-Party Services

This extension integrates with:

  • ChatGPT - Subject to OpenAI's privacy policy
  • ElevenLabs - Subject to ElevenLabs' privacy policy and terms of service
  • Chrome Web Speech API - Subject to Google's privacy policy

Required Permissions

The extension requires the following permissions:

  • Microphone access - For voice input functionality
  • Storage - To save your API key locally
  • Active tab - To interact with ChatGPT's interface
  • webRequest - To handle API communications

Data Security

We implement security measures including:

  • Local storage of sensitive data (API keys)
  • Direct API communications without intermediate servers
  • No persistent storage of voice or text data

Children's Privacy

Our extension is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.
